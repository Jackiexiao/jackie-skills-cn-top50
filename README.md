# Jackie Skills CN Top50

来自 [skills.sh](https://skills.sh) 榜单前 50 的技能集合，已做中文化整理（每个 Skill 的 `title/description`）。

## 一键安装

```bash
npx skills add Jackiexiao/jackie-skills-cn-top50 --all
```

## 分类速览

- 前端与设计：13
- 工程流程与协作：10
- 增长与营销：13
- 文档与办公：4
- 浏览器自动化与测试：4
- 后端与数据：3
- 技能生态与工具链：3
